# FTC Autonomous Builder - Team 23069 (Vercel-ready)

This project is ready to deploy on Vercel (Vite + React).

How to deploy (quick):
1. Unzip the project and make sure package.json is at the root.
2. On Vercel choose: Import Project -> Upload and select this folder.
3. Set Framework to Vite (if not auto-detected).
4. Build Command: npm run build
   Output Directory: dist
5. Deploy!

If you want help during upload, tell me which step you're on.
